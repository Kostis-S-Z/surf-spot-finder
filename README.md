# Surf Spot Finder: a Blueprint for comparing agent frameworks on a specific task

This blueprint demonstrates an AI agent designed for a specific use case: helping users find the next great surf spot near their location.

The agent is provided with input parameters including a location, a distance, and a timestamp. It independently searches and browses the web to recommend the best surf spots based on the given inputs, providing relevant information for decision making.

Although this use-case focuses on surf spots, the underlying framework can be adapted to other agent use cases that require web access and autonomous search.

This implementation utilizes the [smolagents](https://huggingface.co/docs/smolagents/index) library for agent capabilities, and follows the Model Context Protocol (MCP) standard for tool access communication.

## How it Works

The Surf Spot Finder is an AI-driven agent that takes user inputs and autonomously searches the web via integrated tools and models to find surf spots matching the criteria. It gathers and processes relevant information to provide a final recommendation.

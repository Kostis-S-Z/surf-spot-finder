## Setup options



=== "💻 Local Installation"


      1. **Clone the Repository**

         ```bash
         git clone https://github.com/mozilla-ai/surf-spot-finder.git
         cd surf-spot-finder
         ```

      2. **Install the project and its Dependencies**

         ```bash
         pip install -e .
         ```

      3. **Run**

         ```bash
         surf-spot-finder examples/single_agent_with_tools.yaml
         ```
